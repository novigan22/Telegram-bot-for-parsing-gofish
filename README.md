# GoFish Tracker Bot

Телеграм-бот для отслеживания новых товаров на сайте GoFish.

## Структура проекта

```
├── bot.py              # Основной файл бота
├── parser.py           # Парсер GoFish
├── database.py         # База данных
├── requirements.txt    # Зависимости
└── README.md          # Эта инструкция
```

## Установка

1. Установите зависимости:
pip install -r requirements.txt


2. Настройте токен бота и user_id:
   - В файлах `bot.py` и `parser.py` замените `YOUR_BOT_TOKEN_HERE` на токен вашего бота
   - В файле `parser.py` замените `123456789` на ваш Telegram user_id (узнать можно через @userinfobot)

## Запуск

1. Запустите бота:
```bash
python bot.py
```

2. В другом терминале запустите парсер:
```bash
python parser.py
```

## Как использовать

1. Откройте бота в Telegram и нажмите `/start`
2. Нажмите кнопку **"Добавить"** и отправьте ссылку на страницу GoFish
3. Парсер будет проверять эту ссылку каждую минуту
4. При появлении новых товаров вы получите уведомление с фото, названием, ценой и ссылкой

## Функции бота

- ✅ Добавление ссылок для отслеживания
- ✅ Просмотр списка всех ссылок
- ✅ Удаление ссылок
- ✅ Автоматические уведомления о новых товарах
- ✅ Проверка каждую минуту
- ✅ Фильтрация уже показанных товаров

## База данных

Используется SQLite с двумя таблицами:
- `user_links` - хранит ссылки пользователя
- `tracked_products` - хранит ID уже показанных товаров

## Примечания

- Парсер работает отдельно от бота и должен быть запущен в отдельном процессе
- Интервал проверки: 60 секунд
- Товары идентифицируются по уникальному ID
